## Sobre mí

Soy desarrollador Full Stack con experiencia construyendo aplicaciones web orientadas a producto, administración, automatización de procesos y gestión de datos.

He trabajado en plataformas reales para gestión de circos, postulaciones de artistas, sistemas de votación, dashboards administrativos y sitios web multilingües. También me desempeño como **Technical Instructor**, acompañando a estudiantes en el desarrollo de aplicaciones full stack, buenas prácticas, debugging, Git/GitHub y arquitectura de proyectos.

Mi trayectoria profesional combina tecnología, docencia y más de 16 años como artista circense. Esa experiencia me ha dado una forma distinta de enfrentar los problemas: con creatividad, comunicación clara, adaptación, empatía y foco en las personas.

Actualmente estoy abierto a oportunidades remotas o híbridas en España como:

- Full Stack Developer
- Backend Developer

- React Developer
- Ruby on Rails Developer
- Technical Instructor

## 🚀 Proyectos Destacados

### 🎟️ CircusPass

Plataforma de ticketing especializada para circos, enfocada en venta de entradas, control de acceso, reportes y experiencia simple de compra.

**Tecnologías:** TypeScript, React, Node.js, PostgreSQL

---

### 🎪 [CircAdmin](https://github.com/toffycaluga/circadmin)

Plataforma de administración para circos. Permite gestionar ingresos, egresos, usuarios, roles, localidades, planillas y resúmenes financieros.

**Tecnologías:** Ruby on Rails, PostgreSQL, Hotwire, Turbo, Stimulus, Bootstrap, Devise, Cancancan

---

### 🎭 [FestivalApp](https://github.com/toffycaluga/FestivalApp)

Aplicación para gestionar postulaciones de artistas circenses a festivales internacionales, con roles personalizados y dashboards para distintos tipos de usuarios.

**Tecnologías:** Ruby on Rails, PostgreSQL, Devise, Bootstrap

---

### 🌐 [Flying Cáceres](https://github.com/toffycaluga/flying_caceres_)

Sitio web multilingüe para una compañía profesional de trapecistas, con galerías dinámicas, contenido internacionalizado y medios alojados en AWS S3.

**Tecnologías:** Astro, JavaScript, Tailwind CSS, AWS S3, i18n

---

### 🕹️ Toffy's Dev Quest

Portafolio interactivo inspirado en consolas retro, construido con JavaScript, Canvas, CSS e internacionalización.

**Tecnologías:** JavaScript, HTML Canvas, CSS, i18n

---

### 🎪 The Marlon Crew 3D Portfolio

Portafolio 3D inmersivo para una compañía de trapecistas, con ambiente circense, estructuras personalizadas, iluminación dinámica y vistas interactivas.

**Tecnologías:** Astro, JavaScript, Three.js, WebGL
