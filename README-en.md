## 🚀 Featured Projects

### 🎟️ CircusPass

A ticketing platform specialized for circus companies, focused on ticket sales, access control, reporting and a simple purchasing experience.

**Technologies:** TypeScript, React, Node.js, PostgreSQL

---

### 🎪 [CircAdmin](https://github.com/toffycaluga/circadmin)

An administration platform for circus companies. It allows managing income, expenses, users, roles, localities, payrolls and financial summaries.

**Technologies:** Ruby on Rails, PostgreSQL, Hotwire, Turbo, Stimulus, Bootstrap, Devise, Cancancan

---

### 🎭 [FestivalApp](https://github.com/toffycaluga/FestivalApp)

An application for managing circus artist applications to international festivals, with custom roles and dashboards for different user types.

**Technologies:** Ruby on Rails, PostgreSQL, Devise, Bootstrap

---

### 🌐 [Flying Cáceres](https://github.com/toffycaluga/flying_caceres_)

A multilingual website for a professional flying trapeze company, with dynamic galleries, internationalized content and media hosted on AWS S3.

**Technologies:** Astro, JavaScript, Tailwind CSS, AWS S3, i18n

---

### 🕹️ Toffy's Dev Quest

An interactive developer portfolio inspired by retro handheld consoles, built with JavaScript, Canvas, CSS and internationalization.

**Technologies:** JavaScript, HTML Canvas, CSS, i18n

---

### 🎪 The Marlon Crew 3D Portfolio

An immersive 3D portfolio for a flying trapeze company, featuring a circus environment, custom structures, dynamic lighting and interactive views.

**Technologies:** Astro, JavaScript, Three.js, WebGL
